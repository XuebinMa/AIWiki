# 必读 20 条

全书 [79 条](/welcome)太多，不知从哪读起？先读这 20 条。

**怎么选的**（写在前面，免得变成主观清单）：先取 `severity: 高` 的条目，再按两条轴粗排——**新手最早会踩**、**后果最不可逆**。两条都重的排在最前。少数几条严重度虽非「高」，但属于新手第一段会话就会踩、不学就反复吃亏的基础项，也收了进来。读完这 20 条，你已经避开了绝大多数代价最高的坑。

想系统地理解这些坑**为什么**会发生，去 [机制索引](/mechanisms)；想按你的角色或工具浏览，去 [按角色浏览](/roles) 和 [工具矩阵](/tool-matrix)。
