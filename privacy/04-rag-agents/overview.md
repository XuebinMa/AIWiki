## 本卷范围

检索增强与 agent 把数据边界从「一个模型」扩成「一整套系统」，泄露点也随之增多：

- **检索泄露**：向量库 / 知识库的跨租户、跨用户检索，ACL 在检索前执行还是检索后过滤。
- **跨会话 / 跨用户记忆串味**：长期 memory 把一个用户的私有上下文带到另一个会话。
- **工具调用与 agent 外泄**：私有数据被工具发往外部；致命三件套（私有数据 + 不可信内容 + 对外通信）的隐私视角复现。
- **多 agent 协作的数据边界**：subagent 之间共享了不该共享的上下文。
