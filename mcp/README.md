# aiwiki-mcp

MCP server for [AiWiki](https://xuebinma.github.io/AIWiki/) — search and read an encyclopedia of **AI-coding pitfalls** and **LLM privacy protection**, written from the AI's first-person perspective. Bilingual (EN/中文), every entry evidence-tagged (papers / vendor docs / CVEs).

把 AiWiki 百科变成你编辑器里的工具：遇到「AI 越聊越笨」「幻觉 API」「测试被糊弄」这类反复出现的问题时，让你的 AI 助手直接查到对应误区条目、机制解释和工程护栏。

## Tools

| Tool | What it does |
| --- | --- |
| `aiwiki_search` | Search entries by keywords or a short problem description (Chinese or English). Optional `subject` filter (`coding-pitfalls` / `privacy`), `lang`, `limit`. |
| `aiwiki_get_entry` | Fetch the full markdown of one entry by `id` — mechanism analysis, consequences, best practices, and verifiable sources. |

Content is fetched live from the site's public index (`/api/index.json`) and GitHub — the package ships no content and never goes stale.

## Install

### Claude Code

```bash
claude mcp add aiwiki -- npx -y aiwiki-mcp
```

### Other MCP clients (Cursor, Windsurf, Claude Desktop…)

```json
{
  "mcpServers": {
    "aiwiki": {
      "command": "npx",
      "args": ["-y", "aiwiki-mcp"]
    }
  }
}
```

### From a repo checkout (no npm)

```bash
git clone https://github.com/XuebinMa/AIWiki.git
cd AIWiki/mcp && npm install
claude mcp add aiwiki -- node /path/to/AIWiki/mcp/src/server.js
```

## Development

```bash
npm install
npm test          # node:test suites for search & loader
AIWIKI_INDEX_URL=http://localhost:8931/api/index.json npm start   # point at a local index for dev
```

The index is generated by `scripts/gen-mcp-index.js` at the repo root (wired into the site's `prebuild`) and served as `https://xuebinma.github.io/AIWiki/api/index.json`.
